# BEE1117 - Validacao de Nota

## Fluxograma

```mermaid
flowchart TD
A([INICIO]) --> B["somaNotas = 0"]

subgraph SEQ [Laco while]
D -- V --> E[/"nota"/]

H -- LOOP --> D
I -- LOOP --> D
end
D -- F --> J[/"media = somaNotas / 2"/]
```
